## Codebase Patterns
- Backend venv: `backend/.venv/bin/python`
- Django requires `DJANGO_SECRET_KEY=dev-secret-key` env var for manage.py commands
- Run tests: `cd backend && DJANGO_SECRET_KEY=dev-secret-key .venv/bin/python manage.py test core`
- Migrations dir: `backend/core/migrations/`
